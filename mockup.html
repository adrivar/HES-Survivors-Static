Présentation des maquettes

Voici les maquettes visuelles de notre projet. Elles représentent l'apparence graphique de notre application avant son développement.

Start level - Tutoriel

Tuto : capture du tutoriel interactif qui présente les commandes et les mécaniques du jeu (saut, déplacement, interaction avec les modules). Conçu pour guider les nouveaux joueurs.

Exemple level 1 - "Semestre 1"

Les obstacles sont illustrés par les logo des technologies.

Exemple de niveau : aperçu d'un niveau typique dans les couloirs de la HES, avec des obstacles représentant les modules. Le joueur doit traverser, collecter des informations et éviter les pièges.

Écran de défaite

Écran de défaite : illustré lorsqu'un joueur se fait « manger » par un module. Présente les informations du module concerné et propose un quiz ou des options pour recommencer.

Le code source de l'application est disponible sur le github du projet.

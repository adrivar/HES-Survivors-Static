Votre Mission.... Gagner en compétence

http://www.hevs.ch/, Vous êtes un nouvel étudiant en informatique de gestion. Vous allez devoir passer pendant 3 ans, par dessus chaque module à fin d'obtenir votre bachelor. Dans HESsurvivors nous vous proposons de découvrir les différents cours qui vous attendent, grâce à un jeux de style Super Mario Bros. Vous allez devoir traverser les couloirs de la HES, en évitant de vous faire manger par les modules. Si le module vous mange vous recevrez des informations sur le cours en question et un petit quiz vous sera proposé pour continuer votre aventure. Bonne chance et surtout amusez-vous bien !

Le code source de l'application est disponible sur le github du projet.
